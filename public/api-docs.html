📖 Documentație API REST

← Înapoi la App

Introducere

API-ul LittleSteps utilizează arhitectura REST și comunică prin formatul JSON. Autentificarea este gestionată prin sesiuni PHP (Cookies).

🔐 Autentificare

POST /api/auth.php?action=register

Creează un cont nou. Primul utilizator înregistrat primește rolul de admin.

{ "email": "user@test.com", "password": "123", "fullname": "Ion Popescu" }

POST /api/auth.php?action=login

Inițializează o sesiune pentru utilizator.

🍼 Activități (Timeline)

GET /api/feeding.php

Returnează lista tuturor meselor înregistrate pentru utilizatorul curent.

POST /api/sleep.php

Înregistrează un nou eveniment de somn.

{ "details": "Somn de prânz de 2 ore" }

🏥 Medical & Galerie

POST /api/media.php

Încarcă un fișier multimedia. Necesită multipart/form-data.

🛡️ Administrare

GET /api/admin.php

Returnează lista tuturor utilizatorilor. Necesită rol de admin.

Status Codes: 200 OK, 403 Forbidden.

DELETE /api/admin.php?id={userId}

Șterge un utilizator din sistem.

📊 Export & RSS

GET /api/export.php?format=csv

Descarcă toate activitățile în format CSV.

GET /api/rss.php

Generează un flux XML 2.0 cu ultimele 20 de activități.
